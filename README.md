# 🌟 সাথী AI (Sathi AI) — Public Setup & Installation Guide

সাথী AI (Sathi AI) একটি সর্বাধুনিক, স্বয়ংসম্পূর্ণ এবং অত্যন্ত গতিশীল পার্সোনাল অ্যাসিস্ট্যান্ট প্ল্যাটফর্ম। এটি ব্রাউজার ও ডেসক্রিপশনের পাশাপাশি একটি চমৎকার অ্যান্ড্রয়েড অ্যাপ (Android App) হিসেবেও রান করা সম্ভব। 

এই রিপোজিটরিটি শুধুমাত্র কাস্টমার, ব্যবহারকারী এবং ডেভেলপারদের জন্য **সেটআপ এবং ইনস্টলেশন নির্দেশিকা (Implementation Guides)** হিসেবে তৈরি করা হয়েছে। মূল সোর্স কোডটি সম্পূর্ণ **নিরাপদ এবং প্রাইভেট (Private Source Code)** রাখা হয়েছে যাতে আপনার সিকিউরিটি ও আইডেন্টিটি বজায় থাকে।

## ✨ মূল আকর্ষণ ও টেকনোলজির পরিচয়

*   **🤖 ট্রিপল এআই বুস্টার (Triple Model Support):** Gemini 1.5/2.0 এর সাথে ব্যাকআপ হিসেবে ChatGPT (OpenAI) এবং সাশ্রয়ী DeepSeek V3/R1 ইন্টিগ্রেশন।
*   **⚡ ইনস্ট্যান্ট কোয়েরি লোকাল ক্যাশ:** দ্রুত স্পিড ও এপিআই বিল সাশ্রয়ের জন্য শক্তিশালী ২-স্তরের অফলাইন সিম্যান্টিক ক্যাশ মেকানিজম।
*   **📅 দেশীয় বাংলা ক্যালেন্ডার:** গাণিতিক ও বাংলা তিথি গণনা ইঞ্জিন যা রিয়েল টাইমে প্রসেস হয়।
*   **🔒 প্রাইভেট কী এনক্রিপশন:** ব্যবহারকারীরা ব্রাউজার থেকেই সম্পূর্ণ নিজস্ব API Settings কী যুক্ত করতে পারেন যা লোকাল স্টোরেজে নিরাপদে সংরক্ষিত থাকে।
