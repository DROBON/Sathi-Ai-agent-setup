# 🌟 সাথী AI (Sathi AI) — Your Intelligent Personal Companion

সাথী AI একটি সর্বাধুনিক, স্বয়ংসম্পূর্ণ এবং অত্যন্ত গতিশীল পার্সোনাল অ্যাসিস্ট্যান্ট প্ল্যাটফর্ম। এটি ব্রাউজার ও ডেসক্রিপশনের পাশাপাশি একটি চমৎকার অ্যান্ড্রয়েড অ্যাপ (Android App) হিসেবেও ব্যবহার করার জন্য শতভাগ প্রস্তুত। এই ডকুমেন্টটিকে এমনভাবে সাজানো হয়েছে যাতে যে কেউ এই প্রজেক্টের ইনস্টলেশন, কাস্টম এপিআই (API Key) সেটআপ এবং অ্যান্ড্রয়েড বিল্ড প্রোসেস সম্পর্কে পরিষ্কার ধারণা পেতে পারে।

## ✨ মূল আকর্ষণ ও টেকনোলজিক্যাল ফিচারস (Core Features)

*   **🤖 ট্রিপল এআই আর্কিটেকচার (Triple Model Support):** প্রাথমিক মডেল হিসেবে **Google Gemini 1.5/2.0** এর পাশাপাশি ফ্যালব্যাক (কোটা লিমিট এড়ানোর জন্য) হিসেবে **ChatGPT (OpenAI)** এবং বিশ্বের অন্যতম জনপ্রিয় ও সাশ্রয়ী **DeepSeek V3/R1** মডেল ব্যাকআপ হিসেবে সংযুক্ত।
*   **⚡ অফলাইন সিম্যান্টিক ক্যাশ ও সুপারফাস্ট রেসপন্স:** ২ স্তরের হাইব্রিড লোকাল ক্যাশিং মেকানিজম, যার ফলে প্রায়শই জিজ্ঞাসিত প্রশ্নের উত্তর কোনো রিকোয়েস্ট পাঠানো ছাড়াই instantly পাওয়া যায়।
*   **📅 গাণিতিক ও বাংলা ক্যালেন্ডার ইঞ্জিন:** দেশীয় এবং গাণিতিক হিসাব-নিকাশের জটিল অ্যালগরিদম যা ব্রাউজার/ডিভাইসেই ০ মিলি-সেকেন্ডে প্রসেস হয়।
*   **📱 অ্যান্ড্রয়েড ও কম্পিউটার সাপোর্টেড (Cross-Platform):** CapacitorJS লাইব্রেরির মাধ্যমে এটি মোবাইল অ্যাপের আকারে ইনস্টল ও রান করা সম্ভব।
*   **🔒 সর্বোচ্চ ডেটা ও এপিআই নিরাপত্তা:** ব্যবহারকারীরা চাইলে সম্পূর্ণ ফ্রিতে নিজেদের ব্যক্তিগত এপিআই কী যুক্ত করে ব্যবহার করতে পারবেন (ব্রাউজারের লোকাল স্টোরেজে সম্পূর্ণ এনক্রিপ্টেড ও সুরক্ষিত থাকে)।

## 💻 কম্পিউটারে ইনস্টলেশন ও সেটআপ গাইড (Desktop/Local Setup)

আপনার কম্পিউটারে সাথী AI রান করতে নিচের ধাপগুলো অনুসরণ করুন:

### ১. পূর্বশর্ত (Prerequisites)
আপনার কম্পিউটারে অবশ্যই **Node.js** (সংস্করণ ১৮ বা তার বেশি) ইনস্টল করা থাকতে হবে। [ডাউনলোড লিংক](https://nodejs.org/)

### ২. ডিপেন্ডেন্সি ইনস্টলেশন (Install Dependencies)
প্রজেক্টের ডিরেক্টরিতে কমান্ড প্রম্পট বা টার্মিনাল ওপেন করে রান করুন:
```bash
npm install
```

### ৩. এনভায়রনমেন্ট কনফিগারেশন (.env Setup)
রুট ডিরেক্টরিতে থাকা `.env.example` ফাইলটিকে রিনেম (Rename) করে `.env` নাম দিন। এরপর ফাইলের ভেতরে আপনার কাঙ্ক্ষিত API Key সমূহ যুক্ত করুন:
```env
# Google Gemini Setup
GEMINI_API_KEY="আপনার_জেমিনি_এপিআই_কী"
GEMINI_API_KEY_POOL="ঐচ্ছিক_কী_১,ঐচ্ছিক_কী_২"

# OpenAI Setup (Optional Fallback)
OPENAI_API_KEY="sk-..."
OPENAI_API_KEY_POOL="sk-...,sk-..."

# DeepSeek Setup (Optional Fallback)
DEEPSEEK_API_KEY="sk-..."
DEEPSEEK_API_KEY_POOL="sk-...,sk-..."
```

### ৪. ডেভেলপমেন্ট সার্ভার চালু করা (Run App)
লোকাল ডেভেলপমেন্ট সার্ভার চালু করতে এবং ব্রাউজারে প্রিভিউ দেখতে নিচের কমান্ডটি রান করুন:
```bash
npm run dev
```
সার্ভার চালু হলে ব্রাউজারে `http://localhost:3000` (অথবা টার্মিনালে প্রদর্শিত পোর্টে) আপনার নিজের তৈরি অতি প্রিমিয়াম সাথী AI দেখতে পাবেন।

## 🐳 ডকার সেটআপ এবং রান করার নিয়ম (Docker Setup & Installation)

সাথী AI প্রজেক্টটি ডকারের সাহায্যে সম্পূর্ণ আইসোলেটেড এনভায়রনমেন্টে এক ক্লিকে রান করার জন্য ডকার ফাইল ও কম্পোজ কনফিগারেশন সম্পূর্ণরূপে প্রস্তুত করা আছে।

### ১. পূর্বশর্ত (Prerequisites)
আপনার কম্পিউটারে অবশ্যই **Docker** এবং **Docker Compose** ইনস্টল করা থাকতে হবে। [ডাউনলোড লিংক](https://www.docker.com/products/docker-desktop/)

### ২. ডকারের মাধ্যমে সম্পূর্ণ প্রজেক্ট রান করার নিয়ম (Docker Compose)
রুট ডিরেক্টরিতে টার্মিনাল বা কমান্ড প্রম্পট ওপেন করে নিচের কমান্ডটি রান করুন:
```bash
docker-compose up --build -d
```
*এই কমান্ডটি আপনার প্রজেক্টের রিয়্যাক্ট ও এক্সপ্রেস সার্ভার বিল্ড করবে এবং সহযোগী ব্যাকগ্রাউন্ড সার্ভিস সমূহ (যেমন: গ্রাফানা, প্রোমেথিয়াস ইত্যাদি) ব্যাকগ্রাউন্ডে চালু করে দিবে।*

### ৩. শুধুমাত্র সাথী AI সার্ভার রান করার নিয়ম (Single Container Build)
আপনি যদি অন্য কোনো ব্যাকগ্রাউন্ড টুল ছাড়া শুধুমাত্র সাথী AI এর লাইট-ওয়েট কন্টেইনার আলাদাভাবে রান করতে চান, তবে রান করুন:
```bash
# ইমেজ বিল্ড করার জন্য
docker build -t sathi-ai-app .

# কন্টেইনার রান করার জন্য (Port 3000 এ এক্সেস করা যাবে)
docker run --env-file .env -p 3000:3000 -d sathi-ai-app
```

### ৪. ডকার কন্টেইনার স্টপ করার নিয়ম
কন্টেইনারের রান সাময়িকভাবে বন্ধ করতে চাইলে রান করুন:
```bash
docker-compose down
```

## 📱 অ্যান্ড্রয়েড অ্যাপ প্রস্তুত করার নিয়ম (Android Setup & Build)

সাথী AI-কে পুরোপুরি মোবাইলে ইনস্টল ও টেস্ট করার জন্য আমরা **CapacitorJS** যুক্ত করেছি। 

### অ্যান্ড্রয়েড প্রজেক্টের পূর্বশর্ত (Prerequisites for Mobile Build)
1.  **JDK 17** ইনস্টল করুন।
2.  আপনার কম্পিউটারে **Android Studio** ডাউনলোড ও ইনস্টল করুন। [ডাউনলোড লিংক](https://developer.android.com/studio)

---

### ক) মোবাইল রিলিজ বান্ডেল (.aab) তৈরি করা (Play Store এ আপলোড এর জন্য):
গুগল প্লে স্টোরে আপলোড করার জন্য আধুনিক পদ্ধতি হলো অ্যান্ড্রয়েড অ্যাপ বান্ডেল (`.aab` ফাইল)। এটি তৈরি করতে টার্মিনালে এক লাইনের এই সহজ কমান্ডটি রান করুন:
```bash
npm run android:build-bundle
```
*এই স্ক্রিপ্টটি স্বয়ংক্রিয়ভাবে React সোর্স কোড কম্পাইল করবে, মোবাইল এসেট সিঙ্ক করবে এবং রিলিজ বান্ডল তৈরি করবে।*

---

### খ) সরাসরি মোবাইলে টেস্ট করতে APK তৈরি করা:
আপনার নিজের ফোনে সরাসরি ইনস্টল করার জন্য আন-সাইন্ড রিলিজ APK ফাইল জেনারেট করতে রান করুন:
```bash
npm run android:build-apk
```
বিল্ড শেষ হলে আপনি রিলিজ APK ফাইলটি পাবেন নিচের ডিরেক্টরিতে:  
`./android/app/build/outputs/apk/release/app-release-unsigned.apk`

---

### গ) Android Studio এর মাধ্যমে সরাসরি ইন্সটলেশন ও রান (সবচেয়ে সহজ):
১.  টার্মিনালে রান করে প্রজেক্টটি এন্ড্রয়েড স্টুডিওতে ওপেন করুন:
    ```bash
    npm run android:open
    ```
২.  আপনার মোবাইলটি USB ক্যাবলের মাধ্যমে কম্পিউটারে যুক্ত করুন এবং মোবাইলের **Developer Options > USB Debugging** অন করুন।
৩.  Android Studio-র উপরে ডানপাশে আপনার মোবাইলের নাম দেখতে পাবেন, সেখানে **Run Device (Green Play Button)** এ ক্লিক করুন। কয়েক সেকেন্ডের মধ্যে চমৎকার মোবাইল অ্যাপটি আপনার ফোনে সরাসরি ইনস্টল এবং ওপেন হবে!

## 🔒 ইউজারদের কাস্টম এপিআই ব্যবহারের গাইড (Direct Client Key Usage)

সাথী AI এর একটি চমৎকার ফিচার হলো যে কোনো ব্যবহারকারী চাইলে কোনো প্রকার লিমিটেশন ছাড়াই এটি ফ্রিতে ব্যবহার করতে পারেন। 

*   অ্যাপের ডানদিকের উপরে **Settings বা চাবি আইকনে 🔑** ক্লিক করুন।
*   সেখানে আপনি আপনার ব্যক্তিগত **Gemini, ChatGPT বা DeepSeek** এর এপিআই কী যুক্ত করার ঘর পাবেন।
*   আপনার কী যুক্ত করার পর এটি সরাসরি আপনার ডিভাইসে সেভ থাকবে। সাথী মেকের অফিশিয়াল সার্ভার লিমিট ব্যবহার না করে এটি সরাসরি আপনার নিজস্ব কোটা ব্যবহার করবে। ফলে ১ লক্ষ ইউজার এক সাথে মেসেজ করলেও আপনার ব্যবহারের গতি থাকবে অত্যন্ত ফাস্ট এবং শতভাগ স্ট্যাবল।
*   এটি সম্পূর্ণ ফ্রি, নিরাপদ এবং সরাসরি ক্লায়েন্ট এন্ডে কাজ করে।
