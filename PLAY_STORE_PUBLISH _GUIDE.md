🚀 সাথী AI (Sathi AI) - গুগল প্লে-স্টোর পাবলিশিং গাইড (Google Play Store Release Master Guide)
আসসালামু আলাইকুম রাসেল ভাই! সাথী AI-কে আপনি যাতে সফলভাবে গুগল প্লে স্টোরে (Google Play Store) রিলিজ করতে পারেন তার জন্য প্রয়োজনীয় সমস্ত মেকানিজম এবং প্রফেশনাল কনফিগারেশন সেটআপ রেডি করে দেওয়া হয়েছে।

নিচে সহজ ভাষায় ধাপে ধাপে সম্পূর্ণ গাইডটি দেওয়া হলো:

🎨 ১. আইকন ও ব্র্যান্ডিং প্রিপারেশন (আমরা যা প্রস্তুত করেছি)
অ্যাপের নাম: গুগল প্লে-স্টোরে এবং ইউজারের ডিসপ্লেতে প্রদর্শনের জন্য অ্যাপটির নাম পরিবর্তন করে আকর্ষণীয় ও প্রিমিয়াম "সাথী AI" সেট করে দেওয়া হয়েছে (strings.xml এবং manifest.json ফাইলে)।
লোগো ও আইকন: আমরা আপনার জন্য একটি অত্যন্ত প্রিমিয়াম এবং ট্রেন্ডি 3D Glassmorphic Glowing Cybernetic Assistant Brain সহ স্টার স্পার্ক লোগো তৈরি করে /public/icon-512.png হিসেবে সেট করেছি। এটি প্লে-স্টোরে আপলোড করার জন্য এবং মোবাইল অ্যাপের ব্রান্ডিং হিসেবে ব্যবহার করার জন্য শতভাগ প্রস্তুত।
💻 ২. রিলিজ ফাইল তৈরির শর্টকাট কমান্ড (New npm Scripts)
আপনার জন্য package.json ফাইলে নতুন দুটি কমান্ড যুক্ত করে দেওয়া হয়েছে, যা থেকে আপনি এক ক্লিকেই সম্পূর্ণ প্রজেক্টের রিলিজ বান্ডেল বিল্ড করে নিতে পারবেন:

ক) Play Store এর জন্য AAB (Android App Bundle) ফাইল তৈরি করতে:
গুগল প্লে-স্টোরে অ্যাপ আপলোডের জন্য গুগল এখন সরাসরি APK এক্সেপ্ট করে না, তারা .aab ফাইল চায়। এটি তৈরি করতে আপনার কম্পিউটারের টার্মিনালে নিচের কমান্ডটি রান করুন:

npm run android:build-bundle
এই কমান্ডটি প্রথমে আপনার React সাইট বিল্ড করবে, সেটি Android প্রজেক্টে কপি ও সিঙ্ক করবে এবং ব্যাকগ্রাউন্ডে রেডিমেড রিলিজ .aab ফাইল জেনারেট করে দেবে।

খ) সরাসরি মোবাইলে টেস্ট করতে রিলিজ APK তৈরি করতে:
আপনার নিজের বা অন্যদের ফোনে সরাসরি ডাউনলোড করে ইনস্টল করার জন্য রিফাইনড রিলিজ APK তৈরি করতে রান করুন:

npm run android:build-apk
এটি রান করার পর আপনি রিলিজ APK ফাইলটি পেয়ে যাবেন /android/app/build/outputs/apk/release/app-release-unsigned.apk ডিরেক্টরিতে।

🛠️ ৩. Android Studio এর মাধ্যমে সিকিউরলি সাইন করা (সবচেয়ে সহজ ও নিরাপদ পদ্ধতি)
যেহেতু প্লে-স্টোরে ফাইল আপলোড করার জন্য একটি নিরাপদ প্রাইভেট কী (Keystore Signature) প্রয়োজন হয় এবং পাসওয়ার্ড সরাসরি কোডে রাখা অনিরীক্ষিত, তাই Android Studio দিয়ে সাইন করা সবচেয়ে সিকিউর এবং স্ট্যান্ডার্ড প্র্যাকটিস।

ধাপ ১: প্রজেক্টটি Android Studio-তে ওপেন করুন
টার্মিনালে রান করুন:

npm run android:open
অথবা আপনার কম্পিউটারের Android Studio রান করে সরাসরি /android ফোল্ডারটি ওপেন করুন।

ধাপ ২: Keystore জেনারেট এবং সাইন করা
Android Studio-র উপরের মেনুবার থেকে Build > Generate Signed Bundle / APK... এ ক্লিক করুন।
Android App Bundle সিলেক্ট করুন (এটি গুগল প্লে কনসোলে আপলোড হবে) এবং Next চাপুন।
Key store path এর নিচে Create new... বাটনে ক্লিক করুন।
আপনার কম্পিউটারের যেকোনো একটি ড্রাইভে ফাইলটি সেভ করে পাসওয়ার্ড দিন এবং বাকি ডিটেইলস (যেমন আপনার নাম, অর্গানাইজেশন) পূরণ করে OK প্রেস করুন। (⚠️ এই .jks ফাইল এবং এর পাসওয়ার্ড অবশ্যই ভালো জায়গায় সেভ রাখুন যাতে ভবিষ্যতে অ্যাপ আপডেট করতে পাসওয়ার্ড না হারায়!)
Build Variant হিসেবে Release সিলেক্ট করে Finish বাটনে ক্লিক করুন।
কয়েক মিনিটের মধ্যে Android Studio আপনার প্লে-স্টোরে আপলোড যোগ্য সাইন্ড AAB ফাইলটি জেনারেট করে স্ক্রিনে পপআপ দেখাবে "Generate Signed Bundle: Bundle exported successfully"। সেখানে locate বাটনে ক্লিক করলেই আপনার ফাইলটি ডিরেক্টরি থেকে পেয়ে যাবেন!
